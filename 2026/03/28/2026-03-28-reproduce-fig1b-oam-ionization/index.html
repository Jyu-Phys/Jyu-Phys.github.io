快速开始清单

在开始前，请确认已准备以下内容：

  • 物理参数：激光强度 $I=10^{14},\text{W/cm}^2$，波长 800 nm，束腰 $w_0=3,\mu\text{m}$
  • 靶原子：氩原子电离势 $I_p=15.6,\text{eV}=0.5737,\text{a.u.}$，3p 轨道
  • 光束参数：l = ±1，δ = 0（对称），线偏振 s = 0
  • 计算网格：x, y ∈ [-2w₀, 2w₀]，步长 Δx = Δy = 0.1w₀（约 100×100 网格）
  • 关键常数：c = 200（电四极增强因子），光速 $c = 2.998 \times 10^8,\text{m/s}$

执行顺序：A → B → C → D，每个模块可独立验证，模块间通过标准化接口传递数据。

二、模块 A：光场构建（坐标空间）

A1. 对称 LG 光束电场（核心公式）

在焦点处（z = 0），电场分量简化为：

$$E_{\pm,l}(x,y) = E_0 \cdot \left(\frac{\sqrt{2}}{w_0}\right)^{|l|} \cdot \frac{1}{2}\left(x \pm iy\right)^{|l|} \cdot \exp!\left(-\frac{x^2+y^2}{w_0^2}\right)$$

A2. 场梯度计算（电四极项）

电场梯度通过对上式解析求导得到：

$$\frac{\partial E}{\partial x} = E \cdot \left(\frac{|l|}{x \pm iy} - \frac{2x}{w_0^2}\right)$$

A3. 磁场计算（磁偶极项）

线偏振 LG 光束的磁场：

$$B_x = -\frac{i}{\omega_0} \frac{\partial E}{\partial y}, \quad B_y = \frac{i}{\omega_0} \frac{\partial E}{\partial x}$$

✅ 模块 A 验证要点

  1. 电场强度检查：中心点电场最大值应满足 $|E_0| \approx 2.74 \times 10^{10},\text{V/m}$
  2. 强度分布：$|E|^2$ 的峰值强度应等于输入的 $10^{14},\text{W/cm}^2$
  3. 拓扑荷验证：l = +1 时，电场绕中心一周相位变化 $+2\pi$；l = -1 时为 $-2\pi$
  4. 梯度量级：在 $r \sim w_0$ 处，$|\nabla E|/|E| \sim 1/w_0 \approx 3.3 \times 10^5,\text{m}^{-1}$

三、模块 B：原子结构（动量空间）

B1. 氩原子 3p 轨道波函数

动量空间波函数由径向部分和角向球谐函数构成：

$$\widetilde{\psi}g^{(m=\pm1)}(\boldsymbol{\pi}p) = F{3,1}(\pi_p) \cdot Y{1,\pm1}(\theta,\varphi) \propto (\pi_{px} \pm i\pi_{py}) \cdot \frac{F_{3,1}(\pi_p)}{\pi_p}$$

B2. 鞍点动量（Ionization momentum）

在鞍点近似下，动量固定为虚数值：

$$\pi_p = i\sqrt{2I_p}$$

✅ 模块 B 验证要点

  1. 归一化检查：$\int |\widetilde{\psi}_g(\mathbf{p})|^2 d^3p = 1$（在离散网格上数值验证）
  2. 动量尺度：$|\pi_p| = \sqrt{2I_p} \approx 1.07,\text{a.u.}$（对应动能 $I_p$）
  3. 球谐正交性：对 m = 0, ±1 三个态，验证 $\sum_m |Y_{1,m}|^2 = \text{常数}$（与角度无关）

四、模块 C：鞍点动力学

C1. 鞍点方程求解

鞍点时刻 $t_s$ 由以下方程确定（复数解）：

$$\frac{1}{2}\left[\mathbf{p} + \mathbf{A}(t_s)\right]^2 + I_p = 0$$

对于线偏振光和 $\mathbf{p} = i\sqrt{2I_p},\hat{\mathbf{e}}$，解析解为：

$$t_s = \frac{i}{\omega_0} \text{arcsinh}!\left(\frac{\sqrt{2I_p}}{E_0/\omega_0}\right)$$

8.3 与文献对比基准

  1. 平均产额比：$\bar{P}{l=+1} / \bar{P}{l=-1} = 1.00 \pm 0.05$（两图平均强度相等）
  2. 角分布不对称度：$A = (P_{\text{up}} - P_{\text{down}}) / (P_{\text{up}} + P_{\text{down}}) \approx 0.1–0.3$（取决于半径）

  4. 半高全宽：环形峰 FWHM $\approx 0.5 w_0$
